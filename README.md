# Sistema de Gestión de Cursos Académicos

Proyecto académico desarrollado en Java utilizando NetBeans y base de datos.  
El sistema permite registrar, consultar y organizar información relacionada con cursos académicos.

## Tecnologías utilizadas

- Java
- NetBeans IDE
- Base de datos
- JDBC

## Funcionalidades

- Registro de cursos académicos
- Consulta de información de cursos
- Gestión de datos relacionados con cursos
- Conexión con base de datos
- Organización de información académica

## Objetivo del proyecto

El objetivo de este proyecto fue aplicar conocimientos de programación en Java y manejo de base de datos, desarrollando un sistema que permita administrar información de cursos académicos de manera ordenada.

## Base de datos

El proyecto utiliza una base de datos para almacenar la información del sistema.  
Si el repositorio incluye un archivo `.sql`, este puede usarse para crear la estructura de la base de datos.
